# 基于卷积神经网络的图片分类

## 项目介绍
    此数据集包含鸡、兔子和老鼠共三种动物，可分作哺乳纲和鸟纲。数据集中，鸡和兔子各340张图片，老鼠的图片290张。  
    项目共分为三个部分，第一部分为针对哺乳纲（老鼠和兔子）和鸟纲的二分类任务；第二部分为针对鸡、兔子和老鼠的三分类任务；第三阶段为同时针对纲和种进行的多标签分类任务。
